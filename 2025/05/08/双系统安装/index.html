如何在Windows电脑上安装双系统（Windows + Linux）

一、准备工作

1. 所需工具

  • 8GB以上U盘（制作启动盘）
  • Rufus（启动盘制作工具）
  • Linux发行版ISO镜像（推荐Ubuntu/CentOS）
  • 备份重要数据！

2. 磁盘分区

  1. 按 Win+X → 选择”磁盘管理”
  2. 右键C盘 → 选择”压缩卷”
  3. 建议分配至少50GB空间（输入压缩空间量以MB计算）

二、制作启动盘

  1. 打开Rufus工具
  2. 插入U盘（会被格式化）
  3. 选择下载好的ISO文件
  4. 分区类型选择”GPT”（UEFI电脑）或”MBR”（传统BIOS）
  5. 点击”开始”等待完成

三、安装Linux系统

  1. 插入启动盘 → 重启电脑
  2. 开机时按F12/DEL（不同品牌按键不同）进入BIOS
  3. 选择U盘启动

安装步骤示例（Ubuntu）：

  1. 选择”Install Ubuntu”

  2. 语言选择中文 → 继续

  3. 选择”安装类型”时务必选：
    [✓] 与其他操作系统共存
    [✓] 安装时下载更新

  4. 分区设置（关键步骤）：
    • 选择之前压缩的未分配空间

    • 建议分区方案：

    / : 30GB（根目录）
    swap : 内存大小的1-2倍
    /home : 剩余所有空间（用户文件）

  5. 设置用户名密码

  6. 点击”现在安装” → 等待完成
